# harry-potter-sorting-hat

The real Sorting Hat isn't here right now, but this web app can randomly assign your house instead.

Code was written by Claude model ID `claude-sonnet-5` and reviewed by me.

## How to run locally

No build step or dependencies required.

Open the file directly in a browser:
open index.html

Or serve it locally:
python3 -m http.server 8000
Then visit http://localhost:8000
